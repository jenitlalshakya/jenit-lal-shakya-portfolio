## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/jenitlalshakya/jenit-lal-shakya-portfolio.git
cd jenit-lal-shakya-portfolio
Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```
